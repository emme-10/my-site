# Emme's Portfolio

Welcome to Emme's personal portfolio website! This project showcases her design work, philosophy, and contact information.

## Project Structure

```
emme-portfolio
├── index.html        # Main entry point of the website
├── about.html        # Information about Emme and her design philosophy
├── work.html         # Portfolio of Emme's design projects
├── contact.html       # Contact form and information
├── css
│   ├── reset.css     # Resets default browser styles
│   ├── typography.css # Typography styles for the website
│   ├── layout.css    # Layout management for responsive design
│   └── styles.css    # Additional styles for visual appeal
├── js
│   └── main.js       # JavaScript for interactive elements
├── fonts
│   └── .gitkeep      # Placeholder for future font files
├── assets
│   └── .gitkeep      # Placeholder for future assets
└── README.md         # Documentation for the project
```

## Setup Instructions

1. Clone the repository to your local machine.
2. Open the `index.html` file in your web browser to view the website.
3. Customize the content in `about.html`, `work.html`, and `contact.html` as needed.

## Technologies Used

- HTML5
